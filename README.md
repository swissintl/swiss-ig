## 📋 專案概述

這是為**瑞士國際抗衰老（集團）控股有限公司**打造的品牌形象官網，採用多頁式架構設計，清晰區分 B2B（商業合作）與 B2C（消費者）兩大用戶群體。網站風格高端、簡潔，具備瑞士醫學美容與抗衰老品牌的專業感。

### 核心特色
- ✅ **多頁式架構**：非單頁滾動，每個功能獨立頁面
- ✅ **B2B / B2C 雙軌分流**：清晰的用戶導向設計
- ✅ **品牌形象展示**：不含購物車功能，專注品牌呈現
- ✅ **響應式設計**：完美適配桌面、平板、手機
- ✅ **高端視覺風格**：象牙白 + 香檳金 + 深霧灰配色方案
- ✅ **繁體中文為主**：符合香港及大中華市場

## 🚀 使用方式

### 本地開發
1. 直接開啟 `index.html` 即可在瀏覽器中預覽
2. 建議使用 Live Server 或類似工具以獲得更好的開發體驗

### 部署上線
1. 將所有檔案上傳至網站主機
2. 確保保持檔案結構不變
3. 設定 `index.html` 為網站入口
4. 測試所有頁面與連結是否正常運作

## ⚠️ 注意事項

### 表單功能
目前表單提交為**前端模擬**，實際部署時需要：
1. 設定後端 API 接收表單數據
2. 配置電子郵件通知系統
3. 或使用第三方表單服務（如 Google Forms、Formspree）

### 產品圖片
目前產品圖片為 **SVG 佔位符**，實際部署時應：
1. 替換為真實產品照片
2. 優化圖片尺寸（建議 WebP 格式）
3. 使用 CDN 加速圖片載入

### 媒體報導 Logo
目前為文字佔位符，建議替換為：
1. 媒體官方 Logo 圖片
2. 或取得媒體授權的圖片素材
